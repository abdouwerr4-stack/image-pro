# نظام Premium - دليل التطوير

## المعلومات الأساسية:

### الخطط المتاحة:
1. **الحر (Free)** - مجاني
   - التحويل الأساسي
   - صيغ شائعة
   - جودة معيارية

2. **البريميوم (Premium)** - 9.99 ريال/شهرياً
   - كل مميزات الحر
   - مميزات إضافية (قيد التطوير)
   - جودة عالية جداً
   - دعم أولويات

3. **البروفشنال (Pro)** - 19.99 ريال/شهرياً
   - كل مميزات البريميوم
   - مميزات حصرية (قيد التطوير)
   - أولويات عالية
   - دعم مباشر

---

## كيفية إضافة المميزات:

### 1. في converter.html:
```javascript
// تحقق من اشتراك المستخدم
const subscription = JSON.parse(localStorage.getItem('userSubscriptions') || '{}');

if (subscription.premium) {
    // فعّل مميزات البريميوم
    showPremiumFeatures();
}

if (subscription.pro) {
    // فعّل مميزات البريميوم والبروفشنال
    showProFeatures();
}
```

### 2. إضافة زر الترقية:
```html
<button onclick="showUpgradeModal()">ترقية للحصول على هذه الميزة</button>
```

### 3. تتبع الاستخدام:
تُخزن بيانات الاشتراك في `localStorage` تحت:
```
userSubscriptions = {
    premium: { plan, subscribed_at, status },
    pro: { plan, subscribed_at, status }
}
```
